Dictionary Web App

Using the DictionaryAPI this website allows you to search for every word that is available.

Each word can also point out:
  • Synonyms
  • Antonyms
  • Phonetic Audio

You can also switch between light and dark mode. By default the website will pick your beforehand prefered color scheme.

You can choose between three fonts: Arial, Courier, and Libre Baskerville.

Technologies used:

React Typescript SCSS Dictionary API

Forest of An-Ylthin

The forest of An-Ylthin has been usurped by some undead lady. Defeat the lady and all her minions in order to win.

Game features are:
  • Two playable races: dryeldar and undead.
  • Game is turn-based. Multiple units per battle.
  • Four locations need to be cleared in order to win.

Units can use spells and abilities of Combat, Destruction, Dark Magic and Necromancy.

Technologies used:

React React Context Typescript SCSS

City of Hithair

Hithair has turned to turmoil when a wanderer broke through the cemetery and down the mausoleum. After his incursion beneath, sounds and screeches were heard, people got killed, and voices echoed underground. Find the secret below the earth.

Game features are:
  • Two eligible heroes: a Knight and a Dark Wizard.
  • 1v1 turn-based game.
  • Five levels.

There are a merchant that sells power-ups and a skill, and a NPC called Marion that provides a bit of information.

ARC Fiction

ARC Fiction is a movie/tv-show searcher that uses MovieDB API. Home page shows Trending this week, Popular, Upcoming, and Toprated, movies.

Features:
  • Can search for both movies and tv-shows.
  • You can also search for movies or tv-shows based on any category you may like to see.
  • You can make your own lists and delete them.

Every movie or tv-show has a description, trailer, list of similar movies/tv-shows, the top cast, and images.

Technologies used:

React React Context Typescript SCSS
